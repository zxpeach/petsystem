# iwork cloud
一款公司管理软件，包括个人中心、考勤、日程、项目、财务、系统六大功能模块，使用 Vue + Element-ui 实现。
## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
